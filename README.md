#  0x11. C - printf

![Holberton logo](https://www.holbertonschool.com/holberton-logo.png "Holberton Logo")

This project contains the code to create and run a functional copy of printf

## First steps

These instructions allow you to get a copy of the project running on your local machine for development and testing purposes.
```
git clone https://github.com/Gdsoto/printf.git
```

## Developed with
- Programming languaje: C.
- Editor: Vim and Emacs
- Compiler: gcc 4.8.4
## How to use
After cloning the repository, we will use gcc to compile the project.

```
gcc -Wall -Werror -Wextra -pedantic *.c
```

After compiling, there will be an **a.out** file and we execute it with the command.

```
./a
```
